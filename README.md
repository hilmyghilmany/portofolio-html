# Hilmy Portofolio (HTML, CSS, JS)

Personal portfolio website built with plain HTML, CSS, and JavaScript.

## Overview

This project showcases:
- Profile and hero section
- Education timeline
- Experience timeline
- Filterable portfolio gallery (Web Developer / Lighting Designer)
- Portfolio image modal preview
- Light/Dark theme toggle
- Responsive navigation with mobile menu

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- AOS (Animate On Scroll)
- Font Awesome icons

## Project Structure

- [index.html](index.html)
- [style.css](style.css)
- [app.js](app.js)
- [img/](img/)

## Run Locally

1. Open this folder in VS Code.
2. Open [index.html](index.html) in Live Server (or directly in browser).

No build step is required.

## Notes

- Portfolio category tabs include smooth animated transitions.
- Theme is saved in browser `localStorage` (`portfolio-theme`).
